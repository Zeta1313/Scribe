Instillation process:

1. Install Omella at https://ollama.com/download

2. Run Omella pull phi3

3. Open directory and cd into server folder on one console window.

4. Run npm run to start backend

5. Open second console window and run npm run dev in main directory. You should be up and running after that.

Terminal 1:
npm install
npm run dev

Terminal 2:
cd server
npm install
npm run dev
